# 常见危险跨越模式

用于阶段 6 快速生成测试假设。

## 身份类跨越

| 低权限身份 | 高权限目标 | 测试方向 |
|------------|------------|----------|
| 游客 | 认证用户功能 | 未授权访问、Session 伪造 |
| 普通用户 | 其他普通用户数据 | 水平越权、IDOR |
| 普通用户 | VIP/付费功能 | 功能绕过、价格/VIP 校验缺失 |
| 普通用户 | 管理员功能 | 垂直越权、后台接口暴露 |
| 第三方回调 | 内部高权限操作 | 回调伪造、签名绕过 |
| 内部服务 | 用户面数据 | SSRF、内部接口未鉴权 |

## 资产类跨越

| 资产 | 低权限可尝试的操作 | 测试方向 |
|------|--------------------|----------|
| 资金/订单 | 改金额、删字段、负数量、并发 | 0 元购、精度攻击、双花 |
| 优惠券/积分 | 重复使用、叠加、过期后使用 | 羊毛、状态机绕过 |
| 用户资料 | 读/改他人资料 | IDOR、Mass Assignment |
| 文件/文档 | 读/删他人文件、遍历路径 | 文件越权、路径遍历 |
| 管理配置 | 改系统配置、角色权限 | 垂直越权、配置注入 |

## 状态机类跨越

- 订单：unpaid → paid 跳过支付。
- 订单：paid → refunded 绕过退货流程。
- 审核：pending → approved 绕过审核。
- 资格：未满足条件却获得资格。
- 次数限制：突破领取/抽奖/投票次数。

## 第三方/回调类跨越

- 支付回调：伪造 status=success 改订单状态。
- OAuth 回调：篡改 code/state 绑定错误账号。
- 开放重定向：next/redirect 参数指向外部域名。
- Webhook：注册指向内网的回调地址。
